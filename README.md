Welcome to the **Lua Course**!  
This repository is designed to teach Lua from the ground up, covering fundamentals, practical examples, and real-world use cases.

Lua is a **lightweight, fast, and embeddable scripting language**, widely used in:
- Game development (Roblox, Love2D, Defold)
- Embedded systems
- Configuration & scripting
- Extensions for C/C++ applications

- Handle files and errors
- Write clean and reusable Lua code
- Test Lua code using `luaunit`
- Build small practical projects
