# Lua Glossary

- **Chunk**: A piece of Lua code executed by the interpreter
- **Table**: Lua's main data structure
- **Metatable**: A table that defines behavior of another table
- **Coroutine**: Cooperative multitasking mechanism
- **Closure**: Function with captured variables
- **Module**: Reusable Lua code unit
- **pcall**: Protected call for error handling
