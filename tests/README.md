# Lua Tests

This directory contains unit tests for the Lua Course.
All tests are written using `luaunit`.

## Requirements
- Lua 5.4+
- luaunit.lua in your Lua path

## Run all tests
```bash
lua test_basics.lua
lua test_functions.lua
lua test_tables.lua
lua test_modules.lua
lua test_oop.lua
lua test_advanced.lua
